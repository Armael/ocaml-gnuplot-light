Gnuplot-OCaml - Simple interface to Gnuplot
===========================================

---------------------------------------------------------------------------

Gnuplot-OCaml is an [OCaml](http://www.ocaml.org) clone of
[FSharp.GnuPlot](https://github.com/tpetricek/FSharp.GnuPlot)
from Tomas Petricek.
